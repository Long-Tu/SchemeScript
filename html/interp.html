Interacting with an interpreter

SchemeScript provides a minimal interface to Scheme interpreters. There are plans to provide better support in the next releases.

Configuring the interpreter

To define an interpreter, select the Run/External Tools/External Tools... menu item.

Create a new configuration for the interpreter. Give it a meaningful name, like the interpreter's name. Make sure to properly add any command-line argument that rquired by the interpreter (for example, Gambit-C 4.0 requires the -:d- option to run as a child process).

Sending expressions to the interpreter

S-expressions can be sent directly to the interpreter for evaluation using the Eval Expression and Eval toplevel Expression commands from the editor ( à la Emacs). See Editing.
